# Simple-Dart-Calculator-for-Web

To try the calculator:
1. Copy the contents of 'Simple Dart Calculator.dart' to clipboard.
2. Go to https://dartpad.dev/.
3. Paste into the DartPad editor.
4. Click 'run'.
